# Hoop It Up

A PWA (progressive web app) for getting a game going.

## Why a PWA?

PWA's are installable and act like native programs across Windows, Macs, Androids, and iPhones, so running this as a website allows for easy cross-platform access.

## What is it?

Open the website, click yes/no/maybe for whether or not you can play today.
Check back later to see the count of how many have said yes to see if there's a game today.
